# 🚚 Logistics Delivery Performance Analysis

## 📌 Project Overview
SQL-based exploratory analysis of 25,000 logistics delivery records 
to evaluate operational performance, delivery delays, costs, and 
customer satisfaction.

## 🎯 Business Objectives
- Identify factors causing delivery delays and failures
- Compare delivery partner performance
- Assess weather and distance impact on deliveries
- Analyze delivery costs across regions and modes
- Understand customer satisfaction trends

## 🛠️ Tools Used
- PostgreSQL & pgAdmin 4
- Microsoft Excel / CSV
- GitHub

## 📁 Dataset
| Detail | Info |
|--------|------|
| Records | 25,000 |
| Columns | 15 |
| Format | CSV |
| Domain | Logistics & Supply Chain |

## 📊 Key KPIs Analyzed
- Total Deliveries & Delay Percentage
- Failure Rate & Delivery Status Distribution
- Average Delivery Time, Cost & Rating
- Cost per Kilometer & per Kilogram
- Delay % by Weather & Distance Category

## 🔍 SQL Skills Demonstrated
`SELECT` `WHERE` `GROUP BY` `ORDER BY` `CASE WHEN`
`COUNT()` `AVG()` `SUM()` `ROUND()` `RANK()`
Conditional Aggregation | Percentage Calculations | Window Functions

## 💡 Key Insights
1. Weather conditions significantly impacted delivery delays
2. Longer-distance deliveries had higher delay likelihood
3. Delayed deliveries received lower customer ratings
4. Delivery costs varied by region, package type, and mode
5. Some vehicle types consistently delivered faster
